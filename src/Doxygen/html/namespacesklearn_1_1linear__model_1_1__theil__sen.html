42

44 Current start vector.
45
46 Returns
47 -------
48 x_new : ndarray of shape (n_features,)
49 New iteration step.
50
51 References
52 ----------
53 - On Computation of Spatial Median for Robust Data Mining, 2005
54 T. Kärkkäinen and S. Äyrämö
55 http://users.jyu.fi/~samiayr/pdf/ayramo_eurogen05.pdf
56 """
57 diff = X - x_old
58 diff_norm = np.sqrt(np.sum(diff**2, axis=1))
59 mask = diff_norm >= _EPSILON
60 # x_old equals one of our samples
61 is_x_old_in_X = int(mask.sum() < X.shape[0])
62
63 diff = diff[mask]
64 diff_norm = diff_norm[mask][:, np.newaxis]
65 quotient_norm = linalg.norm(np.sum(diff / diff_norm, axis=0))
66
67 if quotient_norm > _EPSILON: # to avoid division by zero
68 new_direction = np.sum(X[mask, :] / diff_norm, axis=0) / np.sum(
69 1 / diff_norm, axis=0
70 )
71 else:
72 new_direction = 1.0
73 quotient_norm = 1.0
74
75 return (
76 max(0.0, 1.0 - is_x_old_in_X / quotient_norm) * new_direction
77 + min(1.0, is_x_old_in_X / quotient_norm) * x_old
78 )

Spatial median (L1 median).

The spatial median is member of a class of so-called M-estimators which
are defined by an optimization problem. Given a number of p points in an
n-dimensional space, the point x minimizing the sum of all distances to the
p other points is called spatial median.

Parameters

    Training vector, where `n_samples` is the number of samples and
    `n_features` is the number of features.

    Maximum number of iterations.

tol : float, default=1.e-3
    Stop the algorithm if spatial_median has converged.

Returns

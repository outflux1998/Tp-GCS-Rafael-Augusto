Detailed Description

Caseless version of L{Keyword}.

Example::
    OneOrMore(CaselessKeyword("CMD")).parseString("cmd CMD Cmd10") # -> ['CMD', 'CMD']
    
(Contrast with example for L{CaselessLiteral}.)

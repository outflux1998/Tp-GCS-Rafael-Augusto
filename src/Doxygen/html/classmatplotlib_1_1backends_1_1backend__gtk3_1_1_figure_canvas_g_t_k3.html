Convert the position of a GTK event, or of the current cursor position
if *event* is None, to Matplotlib coordinates.

GTK use logical pixels, but the figure is scaled to physical pixels for
rendering.  Transform to physical pixels so that all of the down-stream
transforms work as expected.

Also, the origin is different and needs to be corrected.
130 def _mpl_coords(self, event=None):
131 """
132 Convert the position of a GTK event, or of the current cursor position
133 if *event* is None, to Matplotlib coordinates.
134
135 GTK use logical pixels, but the figure is scaled to physical pixels for
136 rendering. Transform to physical pixels so that all of the down-stream
137 transforms work as expected.
138
139 Also, the origin is different and needs to be corrected.
140 """
141 if event is None:
142 window = self.get_window()

144 window.get_display().get_device_manager().get_client_pointer())
145 else:
146 x, y = event.x, event.y

148 # flip y so y=0 is bottom of canvas

150 return x, y

Request a widget redraw once control returns to the GUI event loop.

Even if multiple calls to `draw_idle` occur before control returns
to the GUI event loop, the figure will only be rendered once.

Notes

Backends may choose to override the method and implement their own
strategy to prevent multiple renderings.

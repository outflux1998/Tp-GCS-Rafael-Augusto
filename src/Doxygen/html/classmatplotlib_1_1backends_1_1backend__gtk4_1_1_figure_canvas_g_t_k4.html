Convert the *xy* position of a GTK event, or of the current cursor
position if *xy* is None, to Matplotlib coordinates.

GTK use logical pixels, but the figure is scaled to physical pixels for
rendering.  Transform to physical pixels so that all of the down-stream
transforms work as expected.

Also, the origin is different and needs to be corrected.

91 """
92 Convert the *xy* position of a GTK event, or of the current cursor
93 position if *xy* is None, to Matplotlib coordinates.
94
95 GTK use logical pixels, but the figure is scaled to physical pixels for
96 rendering. Transform to physical pixels so that all of the down-stream
97 transforms work as expected.
98
99 Also, the origin is different and needs to be corrected.
100 """
101 if xy is None:
102 surface = self.get_native().get_surface()
103 is_over, x, y, mask = surface.get_device_position(
104 self.get_display().get_default_seat().get_pointer())
105 else:
106 x, y = xy
107 x = x * self.device_pixel_ratio
108 # flip y so y=0 is bottom of canvas

Set the current cursor.

This may have no effect if the backend does not display anything.

If required by the backend, this method should trigger an update in
the backend event loop after the cursor is set, as this method may be
called e.g. before a long-running task during which the GUI is not
updated.

Parameters
----------

    The cursor to display over the canvas. Note: some backends may
    change the cursor for the entire window.

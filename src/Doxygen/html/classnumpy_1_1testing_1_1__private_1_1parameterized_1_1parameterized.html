Detailed Description

 Parameterize a test case::

        class TestInt:
            @parameterized([
                ("A", 10),
                ("F", 15),
                param("10", 42, base=42)
            ])
            def test_int(self, input, expected, base=16):
                actual = int(input, base=base)
                assert_equal(actual, expected)

        @parameterized([
            (2, 3, 5)
            (3, 5, 8),
        ])
        def test_add(a, b, expected):
            assert_equal(a + b, expected)

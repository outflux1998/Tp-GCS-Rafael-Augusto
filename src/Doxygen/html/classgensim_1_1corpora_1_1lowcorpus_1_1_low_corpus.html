Detailed Description

Corpus handles input in `GibbsLda++ format <http://gibbslda.sourceforge.net/>`_.

**Format description**

Both data for training/estimating the model and new data (i.e., previously unseen data) have the same format
as follows ::

    [M]
    [document1]
    [document2]
    ...
    [documentM]

in which the first line is the total number for documents [M]. Each line after that is one document.
[documenti] is the ith document of the dataset that consists of a list of Ni words/terms ::

    [documenti] = [wordi1] [wordi2] ... [wordiNi]

in which all [wordij] (i=1..M, j=1..Ni) are text strings and they are separated by the blank character.

Examples
--------
.. sourcecode:: pycon

    >>> from gensim.test.utils import get_tmpfile, common_texts
    >>> from gensim.corpora import LowCorpus
    >>> from gensim.corpora import Dictionary
    >>>
    >>> # Prepare needed data
    >>> dictionary = Dictionary(common_texts)
    >>> corpus = [dictionary.doc2bow(doc) for doc in common_texts]
    >>>
    >>> # Write corpus in GibbsLda++ format to disk
    >>> output_fname = get_tmpfile("corpus.low")
    >>> LowCorpus.serialize(output_fname, corpus, dictionary)
    >>>
    >>> # Read corpus
    >>> loaded_corpus = LowCorpus(output_fname)

146 """Covert line into document in BoW format.
147
148 Parameters

156 Document in BoW format
157
158 """
159 words = self.line2words(line)
160
161 if self.use_wordids:
162 # the following creates a unique list of words *in the same order*
163 # as they were in the input. when iterating over the documents,
164 # the (word, count) pairs will appear in the same order as they
165 # were in the input (bar duplicates), which looks better.
166 # if this was not needed, we might as well have used useWords = set(words)
167 use_words, counts = [], Counter()
168 for word in words:
169 if word not in self.word2id:
170 continue
171 if word not in counts:
172 use_words.append(word)
173 counts[word] += 1
174 # construct a list of (wordIndex, wordFrequency) 2-tuples
175 doc = [(self.word2id[w], counts[w]) for w in use_words]
176 else:
177 word_freqs = Counter(words)
178 # construct a list of (word, wordFrequency) 2-tuples
179 doc = list(word_freqs.items())
180
181 # return the document, then forget it and move on to the next one
182 # note that this way, only one doc is stored in memory at a time, not the whole corpus
183 return doc

203 Warnings

213 Corpus in BoW format.
214 id2word : {dict of (int, str), :class:`~gensim.corpora.dictionary.Dictionary`}, optional
215 Mapping between word_ids (integers) and words (strings).
216 If not provided, the mapping is constructed directly from `corpus`.
217 metadata : bool, optional
218 THIS PARAMETER WILL BE IGNORED.
219
220 Return
221 ------
222 list of int
223 List of offsets in resulting file for each document (in bytes),
224 can be used for :meth:`~gensim.corpora.lowcorpus.LowCorpus.docbyoffset`
225
226 """
227 if id2word is None:
228 logger.info("no word id mapping provided; initializing from corpus")
229 id2word = utils.dict_from_corpus(corpus)
230
231 logger.info("storing corpus in List-Of-Words format into %s" % fname)
232 truncated = 0
233 offsets = []
234 with utils.open(fname, 'wb') as fout:
235 fout.write(utils.to_utf8('%i\n' % len(corpus)))
236 for doc in corpus:
237 words = []
238 for wordid, value in doc:
239 if abs(int(value) - value) > 1e-6:
240 truncated += 1
241 words.extend([utils.to_unicode(id2word[wordid])] * int(value))
242 offsets.append(fout.tell())
243 fout.write(utils.to_utf8('%s\n' % ' '.join(words)))
244
245 if truncated:
246 logger.warning(

248 "%i float entries were truncated to integer value", truncated
249 )

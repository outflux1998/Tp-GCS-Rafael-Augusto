Construct BlockManager from string description.

String description syntax looks similar to np.matrix initializer.  It looks
like this::

    a,b,c: f8; d,e,f: i8

Rules are rather simple:

* see list of supported datatypes in `create_block` method
* components are semicolon-separated
* each component is `NAME,NAME,NAME: DTYPE_ID`
* whitespace around colons & semicolons are removed
* components with same DTYPE_ID are combined into single block
* to force multiple blocks with same dtype, use '-SUFFIX'::

    'a:f8-1; b:f8-2; c:f8-foobar'
183def create_mgr(descr, item_shape=None):
184 """
185 Construct BlockManager from string description.
186
187 String description syntax looks similar to np.matrix initializer. It looks
188 like this::
189
190 a,b,c: f8; d,e,f: i8
191
192 Rules are rather simple:
193
194 * see list of supported datatypes in `create_block` method
195 * components are semicolon-separated
196 * each component is `NAME,NAME,NAME: DTYPE_ID`
197 * whitespace around colons & semicolons are removed
198 * components with same DTYPE_ID are combined into single block
199 * to force multiple blocks with same dtype, use '-SUFFIX'::
200
201 'a:f8-1; b:f8-2; c:f8-foobar'
202
203 """
204 if item_shape is None:
205 item_shape = (N,)
206
207 offset = 0
208 mgr_items = []
209 block_placements = {}
210 for d in descr.split(";"):
211 d = d.strip()
212 if not len(d):
213 continue
214 names, blockstr = d.partition(":")[::2]
215 blockstr = blockstr.strip()
216 names = names.strip().split(",")
217
218 mgr_items.extend(names)
219 placement = list(np.arange(len(names)) + offset)
220 try:
221 block_placements[blockstr].extend(placement)
222 except KeyError:
223 block_placements[blockstr] = placement
224 offset += len(names)
225
226 mgr_items = Index(mgr_items)
227
228 blocks = []
229 num_offset = 0
230 for blockstr, placement in block_placements.items():
231 typestr = blockstr.split("-")[0]
232 blocks.append(
233 create_block(
234 typestr, placement, item_shape=item_shape, num_offset=num_offset
235 )
236 )
237 num_offset += len(placement)
238

Detailed Description

Base class for training, using and evaluating \*2vec model.

Contains implementation for multi-threaded training. The purpose of this class is to provide a
reference interface for concrete embedding implementations, whether the input space is a corpus
of words, documents or anything else. At the same time, functionality that we expect to be common
for those implementations is provided here to avoid code duplication.

In the special but usual case where the input space consists of words, a more specialized layer
is provided, consider inheriting from :class:`~gensim.models.base_any2vec.BaseWordEmbeddingsModel`

Notes
-----
A subclass should initialize the following attributes:

* self.kv - keyed vectors in model (see :class:`~gensim.models.keyedvectors.Word2VecKeyedVectors` as example)
* self.vocabulary - vocabulary (see :class:`~gensim.models.word2vec.Word2VecVocab` as example)
* self.trainables - internal matrices (see :class:`~gensim.models.word2vec.Word2VecTrainables` as example)

239 total_words : int, optional
240 Count of total objects in `data_iterator`. In the usual case this would correspond to the number of raw
241 words in a corpus. Used to log progress.
242
243 """
244 job_batch, batch_size = [], 0
245 pushed_words, pushed_examples = 0, 0
246 next_job_params = self._get_job_params(cur_epoch)
247 job_no = 0
248
249 for data_idx, data in enumerate(data_iterator):
250 data_length = self._raw_word_count([data])
251
252 # can we fit this sentence into the existing job batch?
253 if batch_size + data_length <= self.batch_words:
254 # yes => add it to the current job
255 job_batch.append(data)
256 batch_size += data_length
257 else:
258 job_no += 1
259 job_queue.put((job_batch, next_job_params))
260
261 # update the learning rate for the next job
262 if total_examples:
263 # examples-based decay
264 pushed_examples += len(job_batch)
265 epoch_progress = 1.0 * pushed_examples / total_examples
266 else:
267 # words-based decay
268 pushed_words += self._raw_word_count(job_batch)
269 epoch_progress = 1.0 * pushed_words / total_words
270 next_job_params = self._update_job_params(next_job_params, epoch_progress, cur_epoch)
271
272 # add the sentence that didn't fit as the first item of a new job
273 job_batch, batch_size = [data], data_length
274 # add the last job too (may be significantly smaller than batch_words)
275 if job_batch:
276 job_no += 1
277 job_queue.put((job_batch, next_job_params))
278
279 if job_no == 0 and self.train_count == 0:
280 logger.warning(
281 "train() called with an empty iterator (if not intended, "
282 "be sure to provide a corpus that offers restartable iteration = an iterable)."
283 )
284
285 # give the workers heads up that they can finish -- no more work!
286 for _ in range(self.workers):
287 job_queue.put(None)
288 logger.debug("job loop exiting, total %i jobs", job_no)

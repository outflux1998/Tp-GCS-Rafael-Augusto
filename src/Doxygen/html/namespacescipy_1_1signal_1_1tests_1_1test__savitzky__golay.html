This is an alternative implementation of the SG coefficients.

It uses numpy.polyfit and numpy.polyval. The results should be
equivalent to those of savgol_coeffs(), but this implementation

38def alt_sg_coeffs(window_length, polyorder, pos):
39 """This is an alternative implementation of the SG coefficients.
40
41 It uses numpy.polyfit and numpy.polyval. The results should be
42 equivalent to those of savgol_coeffs(), but this implementation
43 is slower.
44
45 window_length should be odd.
46
47 """
48 if pos is None:
49 pos = window_length // 2
50 t = np.arange(window_length)
51 unit = (t == pos).astype(int)
52 h = np.polyval(np.polyfit(t, unit, polyorder), t)
53 return h

Detailed Description

Rosenbrock subject to inequality constraints.

The following optimization problem:
    minimize sum(100.0*(x[1] - x[0]**2)**2.0 + (1 - x[0])**2)
    subject to:  -2 <= x[0] <= 0
                  0 <= x[1] <= 2

Taken from matlab ``fmincon`` documentation.

Detailed Description

TNC non-linear optimization.

These tests are taken from Prof. K. Schittkowski's test examples
for constrained non-linear programming.

http://www.uni-bayreuth.de/departments/math/~kschittkowski/home.htm

328 # gh14879. The output of a TNC minimization was different depending
329 # on whether a callback was used or not. The two should be equivalent.
330 # The issue was that TNC was unscaling/scaling x, and this process was
331 # altering x in the process. Now the callback uses an unscaled

333 def callback(x):
334 pass
335
336 fun = optimize.rosen
337 bounds = [(0, 10)] * 4

339 res = optimize.minimize(
340 fun, x0, bounds=bounds, method="TNC", options={"maxfun": 1000}
341 )
342 res2 = optimize.minimize(
343 fun, x0, bounds=bounds, method="TNC", options={"maxfun": 1000},
344 callback=callback
345 )

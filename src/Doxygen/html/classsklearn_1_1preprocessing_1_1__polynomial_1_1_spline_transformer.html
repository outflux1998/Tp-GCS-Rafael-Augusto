Detailed Description

Generate univariate B-spline bases for features.

Generate a new feature matrix consisting of
`n_splines=n_knots + degree - 1` (`n_knots - 1` for
`extrapolation="periodic"`) spline basis functions
(B-splines) of polynomial order=`degree` for each feature.

Read more in the :ref:`User Guide <spline_transformer>`.

.. versionadded:: 1.0

Parameters
----------
n_knots : int, default=5
    Number of knots of the splines if `knots` equals one of
    {'uniform', 'quantile'}. Must be larger or equal 2. Ignored if `knots`
    is array-like.

degree : int, default=3
    The polynomial degree of the spline basis. Must be a non-negative
    integer.

knots : {'uniform', 'quantile'} or array-like of shape \
    (n_knots, n_features), default='uniform'
    Set knot positions such that first knot <= features <= last knot.

    - If 'uniform', `n_knots` number of knots are distributed uniformly
      from min to max values of the features.
    - If 'quantile', they are distributed uniformly along the quantiles of
      the features.
    - If an array-like is given, it directly specifies the sorted knot
      positions including the boundary knots. Note that, internally,
      `degree` number of knots are added before the first knot, the same
      after the last knot.

extrapolation : {'error', 'constant', 'linear', 'continue', 'periodic'}, \
    default='constant'
    If 'error', values outside the min and max values of the training
    features raises a `ValueError`. If 'constant', the value of the
    splines at minimum and maximum value of the features is used as
    constant extrapolation. If 'linear', a linear extrapolation is used.
    If 'continue', the splines are extrapolated as is, i.e. option
    `extrapolate=True` in :class:`scipy.interpolate.BSpline`. If
    'periodic', periodic splines with a periodicity equal to the distance
    between the first and last knot are used. Periodic splines enforce
    equal function values and derivatives at the first and last knot.
    For example, this makes it possible to avoid introducing an arbitrary
    jump between Dec 31st and Jan 1st in spline features derived from a
    naturally periodic "day-of-year" input feature. In this case it is
    recommended to manually set the knot values to control the period.

include_bias : bool, default=True
    If True (default), then the last spline element inside the data range
    of a feature is dropped. As B-splines sum to one over the spline basis
    functions for each data point, they implicitly include a bias term,
    i.e. a column of ones. It acts as an intercept term in a linear models.

order : {'C', 'F'}, default='C'
    Order of output array. 'F' order is faster to compute, but may slow
    down subsequent estimators.

Attributes
----------
bsplines_ : list of shape (n_features,)
    List of BSplines objects, one for each feature.

n_features_in_ : int
    The total number of input features.

feature_names_in_ : ndarray of shape (`n_features_in_`,)
    Names of features seen during :term:`fit`. Defined only when `X`
    has feature names that are all strings.

    .. versionadded:: 1.0

n_features_out_ : int
    The total number of output features, which is computed as
    `n_features * n_splines`, where `n_splines` is
    the number of bases elements of the B-splines,
    `n_knots + degree - 1` for non-periodic splines and
    `n_knots - 1` for periodic ones.
    If `include_bias=False`, then it is only
    `n_features * (n_splines - 1)`.

See Also
--------
KBinsDiscretizer : Transformer that bins continuous data into intervals.

PolynomialFeatures : Transformer that generates polynomial and interaction
    features.

Notes
-----
High degrees and a high number of knots can cause overfitting.

See :ref:`examples/linear_model/plot_polynomial_interpolation.py
<sphx_glr_auto_examples_linear_model_plot_polynomial_interpolation.py>`.

Examples
--------
>>> import numpy as np
>>> from sklearn.preprocessing import SplineTransformer
>>> X = np.arange(6).reshape(6, 1)
>>> spline = SplineTransformer(degree=2, n_knots=3)
>>> spline.fit_transform(X)
array([[0.5 , 0.5 , 0.  , 0.  ],
       [0.18, 0.74, 0.08, 0.  ],
       [0.02, 0.66, 0.32, 0.  ],
       [0.  , 0.32, 0.66, 0.02],
       [0.  , 0.08, 0.74, 0.18],
       [0.  , 0.  , 0.5 , 0.5 ]])

710 accept_sparse=False,

712 ensure_2d=True,

723 else:

725 if base_knots.shape[0] < 2:
726 raise ValueError("Number of knots, knots.shape[0], must be >= 2.")
727 elif base_knots.shape[1] != n_features:
728 raise ValueError("knots.shape[1] == n_features is violated.")
729 elif not np.all(np.diff(base_knots, axis=0) > 0):
730 raise ValueError("knots must be sorted without duplicates.")
731
732 # number of knots for base interval
733 n_knots = base_knots.shape[0]
734
735 if self.extrapolation == "periodic" and n_knots <= self.degree:
736 raise ValueError(
737 "Periodic splines require degree < n_knots. Got n_knots="
738 f"{n_knots} and degree={self.degree}."
739 )
740
741 # number of splines basis functions
742 if self.extrapolation != "periodic":
743 n_splines = n_knots + self.degree - 1
744 else:
745 # periodic splines have self.degree less degrees of freedom
746 n_splines = n_knots - 1
747
748 degree = self.degree
749 n_out = n_features * n_splines
750 # We have to add degree number of knots below, and degree number knots
751 # above the base knots in order to make the spline basis complete.
752 if self.extrapolation == "periodic":
753 # For periodic splines the spacing of the first / last degree knots
754 # needs to be a continuation of the spacing of the last / first
755 # base knots.
756 period = base_knots[-1] - base_knots[0]
757 knots = np.r_[
758 base_knots[-(degree + 1) : -1] - period,
759 base_knots,
760 base_knots[1 : (degree + 1)] + period,
761 ]
762
763 else:
764 # Eilers & Marx in "Flexible smoothing with B-splines and
765 # penalties" https://doi.org/10.1214/ss/1038425655 advice
766 # against repeating first and last knot several times, which
767 # would have inferior behaviour at boundaries if combined with
768 # a penalty (hence P-Spline). We follow this advice even if our
769 # splines are unpenalized. Meaning we do not:
770 # knots = np.r_[
771 # np.tile(base_knots.min(axis=0), reps=[degree, 1]),
772 # base_knots,
773 # np.tile(base_knots.max(axis=0), reps=[degree, 1])
774 # ]
775 # Instead, we reuse the distance of the 2 fist/last knots.
776 dist_min = base_knots[1] - base_knots[0]
777 dist_max = base_knots[-1] - base_knots[-2]
778
779 knots = np.r_[
780 np.linspace(
781 base_knots[0] - degree * dist_min,
782 base_knots[0] - dist_min,
783 num=degree,
784 ),
785 base_knots,
786 np.linspace(
787 base_knots[-1] + dist_max,
788 base_knots[-1] + degree * dist_max,
789 num=degree,
790 ),
791 ]
792
793 # With a diagonal coefficient matrix, we get back the spline basis
794 # elements, i.e. the design matrix of the spline.
795 # Note, BSpline appreciates C-contiguous float64 arrays as c=coef.
796 coef = np.eye(n_splines, dtype=np.float64)
797 if self.extrapolation == "periodic":
798 coef = np.concatenate((coef, coef[:degree, :]))
799
800 extrapolate = self.extrapolation in ["periodic", "continue"]
801
802 bsplines = [
803 BSpline.construct_fast(
804 knots[:, i], coef, self.degree, extrapolate=extrapolate
805 )
806 for i in range(n_features)
807 ]
808 self.bsplines_ = bsplines
809
810 self.n_features_out_ = n_out - n_features * (1 - self.include_bias)
811 return self

813 def transform(self, X):
814 """Transform each feature data to B-splines.
815
816 Parameters
817 ----------
818 X : array-like of shape (n_samples, n_features)
819 The data to transform.
820
821 Returns
822 -------
823 XBS : ndarray of shape (n_samples, n_features * n_splines)
824 The matrix of features, where n_splines is the number of bases
825 elements of the B-splines, n_knots + degree - 1.
826 """
827 check_is_fitted(self)
828
829 X = self._validate_data(X, reset=False, accept_sparse=False, ensure_2d=True)
830
831 n_samples, n_features = X.shape
832 n_splines = self.bsplines_[0].c.shape[1]
833 degree = self.degree
834
835 # Note that scipy BSpline returns float64 arrays and converts input
836 # x=X[:, i] to c-contiguous float64.
837 n_out = self.n_features_out_ + n_features * (1 - self.include_bias)
838 if X.dtype in FLOAT_DTYPES:
839 dtype = X.dtype
840 else:
841 dtype = np.float64
842 XBS = np.zeros((n_samples, n_out), dtype=dtype, order=self.order)
843
844 for i in range(n_features):
845 spl = self.bsplines_[i]
846
847 if self.extrapolation in ("continue", "error", "periodic"):
848 if self.extrapolation == "periodic":
849 # With periodic extrapolation we map x to the segment
850 # [spl.t[k], spl.t[n]].
851 # This is equivalent to BSpline(.., extrapolate="periodic")
852 # for scipy>=1.0.0.
853 n = spl.t.size - spl.k - 1
854 # Assign to new array to avoid inplace operation
855 x = spl.t[spl.k] + (X[:, i] - spl.t[spl.k]) % (
856 spl.t[n] - spl.t[spl.k]
857 )
858 else:
859 x = X[:, i]
860
861 XBS[:, (i * n_splines) : ((i + 1) * n_splines)] = spl(x)
862
863 else:
864 xmin = spl.t[degree]
865 xmax = spl.t[-degree - 1]
866 mask = (xmin <= X[:, i]) & (X[:, i] <= xmax)
867 XBS[mask, (i * n_splines) : ((i + 1) * n_splines)] = spl(X[mask, i])
868
869 # Note for extrapolation:
870 # 'continue' is already returned as is by scipy BSplines
871 if self.extrapolation == "error":
872 # BSpline with extrapolate=False does not raise an error, but
873 # output np.nan.
874 if np.any(np.isnan(XBS[:, (i * n_splines) : ((i + 1) * n_splines)])):
875 raise ValueError(
876 "X contains values beyond the limits of the knots."
877 )
878 elif self.extrapolation == "constant":
879 # Set all values beyond xmin and xmax to the value of the
880 # spline basis functions at those two positions.
881 # Only the first degree and last degree number of splines
882 # have non-zero values at the boundaries.
883
884 # spline values at boundaries
885 f_min = spl(xmin)
886 f_max = spl(xmax)
887 mask = X[:, i] < xmin
888 if np.any(mask):
889 XBS[mask, (i * n_splines) : (i * n_splines + degree)] = f_min[
890 :degree
891 ]
892
893 mask = X[:, i] > xmax
894 if np.any(mask):
895 XBS[
896 mask,
897 ((i + 1) * n_splines - degree) : ((i + 1) * n_splines),
898 ] = f_max[-degree:]
899
900 elif self.extrapolation == "linear":
901 # Continue the degree first and degree last spline bases
902 # linearly beyond the boundaries, with slope = derivative at
903 # the boundary.
904 # Note that all others have derivative = value = 0 at the
905 # boundaries.
906
907 # spline values at boundaries
908 f_min, f_max = spl(xmin), spl(xmax)
909 # spline derivatives = slopes at boundaries
910 fp_min, fp_max = spl(xmin, nu=1), spl(xmax, nu=1)
911 # Compute the linear continuation.
912 if degree <= 1:
913 # For degree=1, the derivative of 2nd spline is not zero at
914 # boundary. For degree=0 it is the same as 'constant'.
915 degree += 1
916 for j in range(degree):
917 mask = X[:, i] < xmin
918 if np.any(mask):
919 XBS[mask, i * n_splines + j] = (
920 f_min[j] + (X[mask, i] - xmin) * fp_min[j]
921 )
922
923 mask = X[:, i] > xmax
924 if np.any(mask):
925 k = n_splines - 1 - j
926 XBS[mask, i * n_splines + k] = (
927 f_max[k] + (X[mask, i] - xmax) * fp_max[k]
928 )
929
930 if self.include_bias:
931 return XBS
932 else:
933 # We throw away one spline basis per feature.
934 # We chose the last one.
935 indices = [j for j in range(XBS.shape[1]) if (j + 1) % n_splines != 0]
936 return XBS[:, indices]

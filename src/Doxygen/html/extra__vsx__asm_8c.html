#define vsx_ld   vec_xl

Testing ASM VSX register number fixer 'x<n>'

old versions of CLANG doesn't support x<n> in the inline asm template which fixes register number when using any of the register constraints wa, wd, wf.

xref:

  • https://bugs.llvm.org/show_bug.cgi?id=31837

Detailed Description

Read a SAS XPort format file into a Pandas DataFrame.

Based on code from Jack Cushman (github.com/jcushman/xport).

The file format is defined here:

https://support.sas.com/content/dam/SAS/support/en/technical-papers/record-layout-of-a-sas-version-5-or-6-data-set-in-sas-transport-xport-format.pdf

195def _parse_float_vec(vec):
196 """
197 Parse a vector of float values representing IBM 8 byte floats into
198 native 8 byte floats.
199 """
200 dtype = np.dtype(">u4,>u4")
201 vec1 = vec.view(dtype=dtype)
202 xport1 = vec1["f0"]
203 xport2 = vec1["f1"]
204
205 # Start by setting first half of ieee number to first half of IBM
206 # number sans exponent
207 ieee1 = xport1 & 0x00FFFFFF
208
209 # The fraction bit to the left of the binary point in the ieee
210 # format was set and the number was shifted 0, 1, 2, or 3
211 # places. This will tell us how to adjust the ibm exponent to be a
212 # power of 2 ieee exponent and how to shift the fraction bits to
213 # restore the correct magnitude.
214 shift = np.zeros(len(vec), dtype=np.uint8)
215 shift[np.where(xport1 & 0x00200000)] = 1
216 shift[np.where(xport1 & 0x00400000)] = 2
217 shift[np.where(xport1 & 0x00800000)] = 3
218
219 # shift the ieee number down the correct number of places then
220 # set the second half of the ieee number to be the second half
221 # of the ibm number shifted appropriately, ored with the bits
222 # from the first half that would have been shifted in if we
223 # could shift a double. All we are worried about are the low
224 # order 3 bits of the first half since we're only shifting by
225 # 1, 2, or 3.
226 ieee1 >>= shift
227 ieee2 = (xport2 >> shift) | ((xport1 & 0x00000007) << (29 + (3 - shift)))
228
229 # clear the 1 bit to the left of the binary point
230 ieee1 &= 0xFFEFFFFF
231
232 # set the exponent of the ieee number to be the actual exponent
233 # plus the shift count + 1023. Or this into the first half of the
234 # ieee number. The ibm exponent is excess 64 but is adjusted by 65
235 # since during conversion to ibm format the exponent is
236 # incremented by 1 and the fraction bits left 4 positions to the
237 # right of the radix point. (had to add >> 24 because C treats &
238 # 0x7f as 0x7f000000 and Python doesn't)
239 ieee1 |= ((((((xport1 >> 24) & 0x7F) - 65) << 2) + shift + 1023) << 20) | (
240 xport1 & 0x80000000
241 )
242
243 ieee = np.empty((len(ieee1),), dtype=">u4,>u4")
244 ieee["f0"] = ieee1
245 ieee["f1"] = ieee2
246 ieee = ieee.view(dtype=">f8")
247 ieee = ieee.astype("f8")
248
249 return ieee

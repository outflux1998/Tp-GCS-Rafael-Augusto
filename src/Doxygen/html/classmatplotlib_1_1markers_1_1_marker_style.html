Detailed Description

A class representing marker types.

Instances are immutable. If you need to change anything, create a new
instance.

Attributes
----------
markers : list
    All known markers.
filled_markers : list
    All known filled markers. This is a subset of *markers*.
fillstyles : list
    The supported fillstyles.

Detailed Description

Construct a corpus from a Wikipedia (or other MediaWiki-based) database dump.

Uses multiprocessing internally to parallelize the work and process the dump more quickly.

Notes
-----
If you have the `pattern <https://github.com/clips/pattern>`_ package installed,
this module will use a fancy lemmatization to get a lemma of each token (instead of plain alphabetic tokenizer).

See :mod:`gensim.scripts.make_wiki` for a canned (example) command-line script based on this module.

236 # the wiki markup is recursive (markup inside markup etc)
237 # instead of writing a recursive grammar, here we deal with that by removing
238 # markup in a loop, starting with inner-most expressions and working outwards,
239 # for as long as something changes.
240 text = remove_template(text)
241 text = remove_file(text)
242 iters = 0
243 while True:
244 old, iters = text, iters + 1
245 text = re.sub(RE_P0, '', text) # remove comments
246 text = re.sub(RE_P1, '', text) # remove footnotes
247 text = re.sub(RE_P9, '', text) # remove outside links
248 text = re.sub(RE_P10, '', text) # remove math content
249 text = re.sub(RE_P11, '', text) # remove all remaining tags
250 text = re.sub(RE_P14, '', text) # remove categories
251 text = re.sub(RE_P5, '\\3', text) # remove urls, keep description
252
253 if simplify_links:
254 text = re.sub(RE_P6, '\\2', text) # simplify links, keep description only
255 # remove table markup
256 text = text.replace("!!", "\n|") # each table head cell on a separate line
257 text = text.replace("|-||", "\n|") # for cases where a cell is filled with '-'
258 text = re.sub(RE_P12, '\n', text) # remove formatting lines
259 text = text.replace('|||', '|\n|') # each table cell on a separate line(where |{{a|b}}||cell-content)
260 text = text.replace('||', '\n|') # each table cell on a separate line
261 text = re.sub(RE_P13, '\n', text) # leave only cell content
262 text = re.sub(RE_P17, '\n', text) # remove formatting lines
263
264 # remove empty mark-up
265 text = text.replace('[]', '')
266 # stop if nothing changed between two iterations or after a fixed number of iterations
267 if old == text or iters > 2:
268 break
269
270 if promote_remaining:
271 text = text.replace('[', '').replace(']', '') # promote all remaining markup to plain text
272

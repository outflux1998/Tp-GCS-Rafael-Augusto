Detailed Description

Abstract base class for various loss functions.

Parameters
----------
n_classes : int
    Number of classes.

Attributes
----------
K : int
    The number of regression trees to be induced;
    1 for regression and binary classification;
    ``n_classes`` for multi-class classification.

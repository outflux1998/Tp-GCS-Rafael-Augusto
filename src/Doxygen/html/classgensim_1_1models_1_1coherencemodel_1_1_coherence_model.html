Detailed Description

Objects of this class allow for building and maintaining a model for topic coherence.

Examples
---------
One way of using this feature is through providing a trained topic model. A dictionary has to be explicitly provided
if the model does not contain a dictionary already

.. sourcecode:: pycon

    >>> from gensim.test.utils import common_corpus, common_dictionary
    >>> from gensim.models.ldamodel import LdaModel
    >>> from gensim.models.coherencemodel import CoherenceModel
    >>>
    >>> model = LdaModel(common_corpus, 5, common_dictionary)
    >>>
    >>> cm = CoherenceModel(model=model, corpus=common_corpus, coherence='u_mass')
    >>> coherence = cm.get_coherence()  # get coherence value

Another way of using this feature is through providing tokenized topics such as:

.. sourcecode:: pycon

    >>> from gensim.test.utils import common_corpus, common_dictionary
    >>> from gensim.models.coherencemodel import CoherenceModel
    >>> topics = [
    ...     ['human', 'computer', 'system', 'interface'],
    ...     ['graph', 'minors', 'trees', 'eps']
    ... ]
    >>>
    >>> cm = CoherenceModel(topics=topics, corpus=common_corpus, dictionary=common_dictionary, coherence='u_mass')
    >>> coherence = cm.get_coherence()  # get coherence value

662 """Get average topic and model coherences.
663
664 Parameters

675 coherences = []
676 last_topn_value = min(self.topn - 1, 4)
677 topn_grid = list(range(self.topn, last_topn_value, -5))
678
679 for model_num, topics in enumerate(model_topics):
680 self.topics = topics
681
682 # We evaluate at various values of N and average them. This is a more robust,
683 # according to: http://people.eng.unimelb.edu.au/tbaldwin/pubs/naacl2016.pdf
684 coherence_at_n = {}
685 for n in topn_grid:
686 self.topn = n
687 topic_coherences = self.get_coherence_per_topic()
688
689 # Let's record the coherences for each topic, as well as the aggregated
690 # coherence across all of the topics.
691 # Some of them may be nan (if all words were OOV), so do mean value imputation.
692 filled_coherences = np.array(topic_coherences)
693 filled_coherences[np.isnan(filled_coherences)] = np.nanmean(filled_coherences)
694 coherence_at_n[n] = (topic_coherences, self.aggregate_measures(filled_coherences))
695
696 topic_coherences, avg_coherences = zip(*coherence_at_n.values())
697 avg_topic_coherences = np.vstack(topic_coherences).mean(0)
698 model_coherence = np.mean(avg_coherences)
699 logging.info("Avg coherence for model %d: %.5f" % (model_num, model_coherence))
700 coherences.append((avg_topic_coherences, model_coherence))
701
702 return coherences

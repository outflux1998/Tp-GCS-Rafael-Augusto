Evaluate a cubic spline at the new set of points.

`dx` is the old sample-spacing while `x0` was the old origin. In
other-words the old-sample points (knot-points) for which the `cj`
represent spline coefficients were at equally-spaced points of:

Edges are handled using mirror-symmetric boundary conditions.

530 """Evaluate a cubic spline at the new set of points.
531
532 `dx` is the old sample-spacing while `x0` was the old origin. In
533 other-words the old-sample points (knot-points) for which the `cj`
534 represent spline coefficients were at equally-spaced points of:
535
536 oldx = x0 + j*dx j=0...N-1, with N=len(cj)
537
538 Edges are handled using mirror-symmetric boundary conditions.
539
540 Parameters
541 ----------
542 cj : ndarray
543 cublic spline coefficients
544 newx : ndarray
545 New set of points.
546 dx : float, optional
547 Old sample-spacing, the default value is 1.0.
548 x0 : int, optional
549 Old origin, the default value is 0.
550
551 Returns
552 -------
553 res : ndarray
554 Evaluated a cubic spline points.
555
556 See Also
557 --------
558 cspline1d : Compute cubic spline coefficients for rank-1 array.
559
560 Examples
561 --------
562 We can filter a signal to reduce and smooth out high-frequency noise with
563 a cubic spline:
564
565 >>> import matplotlib.pyplot as plt
566 >>> from scipy.signal import cspline1d, cspline1d_eval
567 >>> rng = np.random.default_rng()
568 >>> sig = np.repeat([0., 1., 0.], 100)
569 >>> sig += rng.standard_normal(len(sig))*0.05 # add noise
570 >>> time = np.linspace(0, len(sig))
571 >>> filtered = cspline1d_eval(cspline1d(sig), time)
572 >>> plt.plot(sig, label="signal")
573 >>> plt.plot(time, filtered, label="filtered")
574 >>> plt.legend()
575 >>> plt.show()
576
577 """
578 newx = (asarray(newx) - x0) / float(dx)
579 res = zeros_like(newx, dtype=cj.dtype)
580 if res.size == 0:
581 return res
582 N = len(cj)
583 cond1 = newx < 0
584 cond2 = newx > (N - 1)
585 cond3 = ~(cond1 | cond2)
586 # handle general mirror-symmetry
587 res[cond1] = cspline1d_eval(cj, -newx[cond1])
588 res[cond2] = cspline1d_eval(cj, 2 * (N - 1) - newx[cond2])
589 newx = newx[cond3]
590 if newx.size == 0:
591 return res
592 result = zeros_like(newx, dtype=cj.dtype)
593 jlower = floor(newx - 2).astype(int) + 1
594 for i in range(4):
595 thisj = jlower + i
596 indj = thisj.clip(0, N - 1) # handle edge cases
597 result += cj[indj] * cubic(newx - thisj)
598 res[cond3] = result
599 return res

Gaussian approximation to B-spline basis function of order n.

Parameters
----------
x : array_like
    a knot vector
n : int
    The order of the spline. Must be non-negative, i.e., n >= 0

Returns
-------
res : ndarray
    B-spline basis function values approximated by a zero-mean Gaussian
    function.

Notes
-----
The B-spline basis function can be approximated well by a zero-mean
Gaussian function with standard-deviation equal to :math:`\sigma=(n+1)/12`
for large `n` :

.. math::  \frac{1}{\sqrt {2\pi\sigma^2}}exp(-\frac{x^2}{2\sigma})

References
----------
.. [1] Bouma H., Vilanova A., Bescos J.O., ter Haar Romeny B.M., Gerritsen
   F.A. (2007) Fast and Accurate Gaussian Derivatives Based on B-Splines. In:
   Sgallari F., Murli A., Paragios N. (eds) Scale Space and Variational
   Methods in Computer Vision. SSVM 2007. Lecture Notes in Computer
   Science, vol 4485. Springer, Berlin, Heidelberg
.. [2] http://folk.uio.no/inf3330/scripting/doc/python/SciPy/tutorial/old/node24.html

Examples
--------
We can calculate B-Spline basis functions approximated by a gaussian
distribution:

>>> from scipy.signal import gauss_spline, bspline
>>> knots = np.array([-1.0, 0.0, -1.0])
>>> gauss_spline(knots, 3)
array([0.15418033, 0.6909883, 0.15418033])  # may vary

>>> bspline(knots, 3)
array([0.16666667, 0.66666667, 0.16666667])  # may vary

604
605 Parameters
606 ----------
607 cj : ndarray
608 Quadratic spline coefficients
609 newx : ndarray
610 New set of points.
611 dx : float, optional
612 Old sample-spacing, the default value is 1.0.
613 x0 : int, optional
614 Old origin, the default value is 0.
615
616 Returns
617 -------
618 res : ndarray
619 Evaluated a quadratic spline points.
620
621 See Also
622 --------
623 qspline1d : Compute quadratic spline coefficients for rank-1 array.
624
625 Notes
626 -----
627 `dx` is the old sample-spacing while `x0` was the old origin. In
628 other-words the old-sample points (knot-points) for which the `cj`
629 represent spline coefficients were at equally-spaced points of::
630
631 oldx = x0 + j*dx j=0...N-1, with N=len(cj)
632
633 Edges are handled using mirror-symmetric boundary conditions.
634
635 Examples
636 --------
637 We can filter a signal to reduce and smooth out high-frequency noise with
638 a quadratic spline:
639
640 >>> import matplotlib.pyplot as plt
641 >>> from scipy.signal import qspline1d, qspline1d_eval
642 >>> rng = np.random.default_rng()
643 >>> sig = np.repeat([0., 1., 0.], 100)
644 >>> sig += rng.standard_normal(len(sig))*0.05 # add noise
645 >>> time = np.linspace(0, len(sig))
646 >>> filtered = qspline1d_eval(qspline1d(sig), time)
647 >>> plt.plot(sig, label="signal")
648 >>> plt.plot(time, filtered, label="filtered")
649 >>> plt.legend()
650 >>> plt.show()
651
652 """
653 newx = (asarray(newx) - x0) / dx
654 res = zeros_like(newx)
655 if res.size == 0:
656 return res
657 N = len(cj)
658 cond1 = newx < 0
659 cond2 = newx > (N - 1)
660 cond3 = ~(cond1 | cond2)
661 # handle general mirror-symmetry
662 res[cond1] = qspline1d_eval(cj, -newx[cond1])
663 res[cond2] = qspline1d_eval(cj, 2 * (N - 1) - newx[cond2])
664 newx = newx[cond3]
665 if newx.size == 0:
666 return res
667 result = zeros_like(newx)
668 jlower = floor(newx - 1.5).astype(int) + 1
669 for i in range(3):
670 thisj = jlower + i
671 indj = thisj.clip(0, N - 1) # handle edge cases
672 result += cj[indj] * quadratic(newx - thisj)
673 res[cond3] = result
674 return res

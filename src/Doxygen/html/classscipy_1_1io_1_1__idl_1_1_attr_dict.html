Detailed Description

A case-insensitive dictionary with access via item, attribute, and call
notations:

    >>> d = AttrDict()
    >>> d['Variable'] = 123
    >>> d['Variable']
    123
    >>> d.Variable
    123
    >>> d.variable
    123
    >>> d('VARIABLE')
    123
    >>> d['missing']
    Traceback (most recent error last):
    ...
    KeyError: 'missing'
    >>> d.missing
    Traceback (most recent error last):
    ...
    AttributeError: 'AttrDict' object has no attribute 'missing'
